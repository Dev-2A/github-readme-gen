# 🤖 GitHub 프로필 README 자동 생성기

GitHub API로 활동 데이터(언어 비율, 최근 커밋, 스타 많은 레포 등)를 수집하고,
SVG 배지·차트가 포함된 프로필 README를 자동 생성하는 웹 서비스입니다.

## ✨ 주요 기능

- GitHub 활동 데이터 자동 수집 (언어 비율, 스타, 최근 커밋)
- SVG 배지 및 파이차트 직접 생성
- 파스텔 블루 테마 커스터마이징
- 실시간 미리보기 UI
- 마크다운 복사 / 다운로드

## 🛠 기술 스택

**Backend**: Python, FastAPI, GitHub API  
**Frontend**: React  
**SVG**: 직접 생성 (외부 라이브러리 미사용)
