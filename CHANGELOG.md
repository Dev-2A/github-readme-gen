# Changelog

## [0.1.0] - 2026-03-04

### Added

- GitHub REST API 연동 (유저 정보, 레포, 커밋 이벤트)
- 데이터 파싱 모듈 (언어 비율, 스타 Top 5, 최근 커밋 수)
- SVG 배지 생성기 (프로필 배지, 통계 배지, 라벨 배지)
- SVG 차트 생성기 (파이차트, 바차트, 레포 카드)
- README 마크다운 조립기 (섹션별 커스터마이징)
- FastAPI 백엔드 (에러 핸들링, 응답 모델, 헬스체크)
- React 프론트엔드 (입력 폼, 옵션 패널, 미리보기 UI)
- 파스텔 블루 테마 디자인 시스템
- 스텝 인디케이터, 스켈레톤 로딩, 페이드 인 애니메이션
- 복사 / `.md` 파일 다운로드 기능
